# Hermes

## Project Setup

Copy the repository to your machine

```bash
$ git clone https://github.com/Sir-Aguiar/Hermes.git Hermes
```

Open the directory that've been created

```bash
$ cd Hermes
```

Install all dependencies

```bash
$ npm install
```

Run the development server

```bash
$ npm run dev
```
